# Hybrid-KNX-Modbus-BEMS

This repository contains representative implementation files associated with the research project:

**Hybrid KNX and Modbus RTU Architecture for Intelligent Building Energy Control**

## Repository Contents

### Python_Code

Representative Raspberry Pi scripts used for Modbus RTU monitoring and data acquisition from the DSE7320 generator controller.

### Arduino_Code

Representative Arduino implementations used for distributed KNX-inspired field communication, room-level sensing, and load-control coordination.

### Modbus_Registers

Documentation of the Modbus RTU register mapping used during the experimental implementation.

## Experimental Platform

The developed Building Energy Management System (BEMS) integrates:

* Raspberry Pi supervisory control
* Arduino-based field nodes
* Modbus RTU communication
* KNX-inspired event-driven coordination
* DSE7320 generator monitoring
* Schneider iEM3150 energy metering

## Reproducibility Statement

The repository provides representative implementation materials intended to support transparency, reproducibility, and future research on protocol-integrated Building Energy Management Systems.
